# 📊 GitHub Analytics Dashboard

## 🚀 Overview
A React-based web application that fetches and displays real-time GitHub user data using the GitHub REST API.  
It allows users to search any GitHub profile and view detailed analytics like repositories, followers, and profile information.

---

## 🔗 Live Demo
https://github-dashboard-tawny.vercel.app

---

## ✨ Features
- 🔍 Search GitHub users by username
- 👤 Display user profile information (avatar, bio, followers, following)
- 📦 List public repositories dynamically
- ⭐ Show repository details (stars, forks)
- ⚡ Real-time API integration with GitHub
- 📱 Fully responsive UI

---

## 🛠️ Tech Stack
- React.js
- JavaScript (ES6+)
- GitHub REST API
- Axios / Fetch API
- CSS
- Vercel (Deployment)
